Proposal for application structure. Feel free to modify & add more modules.

- - - - - - - - - - - - - - - - - -

- spectroart.py
Setup & start program

- src/ui.py
Handles user input
Prints & writes output
Calls core.py
Input parameters should be saved here

- src/core.py
Does application logic. Calls imageprocessor.py and helpers.py etc..

- src/imgprocessor.py
Modifies image data
Transforms image to audio with parameters from ui

- src/helpers.py
Random helper functions here
